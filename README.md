# Proyecto: Servidor de Virtualización y Servicios

## 📌 Descripción
Este proyecto consiste en la configuración de un servidor con servicios de red:
- DNS
- Active Directory
- DHCP
- Recursos compartidos

## 🖥️ Estructura del proyecto

- 1-dns → Configuración DNS
- 2-active-directory → Configuración AD
- 3-dhcp → Configuración DHCP
- 4-recursos-compartidos → Carpetas compartidas
- web-demo → Interfaz visual

## ⚙️ Requisitos
- Windows Server
- PowerShell
- VS Code

## 🚀 Ejecución

1. Ejecutar scripts PowerShell como administrador
2. Abrir index.html con Live Server
